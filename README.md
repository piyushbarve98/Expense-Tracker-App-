# Expense-Tracker-App-
Expense Tracker App powered with voice command feature.

 A react expense tracker app which supports speech recognition for quick input. You can enter your incomes/expenses with your voice.i.e no need to type.It will help you to note,delete and categorize your incomes/expenses for easy budget management.
 
 ### Getting started ##
  
  Clone the repository and run `npm i` to install all the dependencies<br>
  Run `npm start` to start the react app<br>
  Navigate to `localhost:3000` on your browser
